# \# Crypto Price Tracker

# 

# A simple web app to track cryptocurrency prices using the CoinGecko API.

# 

# \## Features

# \- Track BTC, ETH, DOGE, SOL

# \- Auto-refresh every 30s

# \- Dark mode toggle

# \- Shows last updated time
